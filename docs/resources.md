## Command Collections

### Version Control
- `/git-branch-cleanup` - Clean up merged branches
- `/git-commit` - Smart commit with message
- `/git-review` - Review changes before commit
- `/github-pr` - Create PR from commits
- `/create-pr` - Generate PR description

### Code Analysis
- `/analyze` - Full codebase analysis
- `/find-code` - Search for patterns
- `/explain-code` - Explain selected code
- `/audit-security` - Security audit

### Development
- `/test` - Generate tests
- `/refactor` - Refactor code
- `/debug` - Debug issues
- `/optimize` - Performance optimization

### Documentation
- `/docs` - Generate documentation
- `/readme` - Update README
- `/changelog` - Generate changelog
- `/api-docs` - Generate API docs

### Project Management
- `/plan` - Create implementation plan
- `/estimate` - Estimate task effort
- `/breakdown` - Break down tasks
- `/track` - Track progress

## Quick Start

### Install Essential Plugins
```bash
/plugin marketplace add hesreallyhim/awesome-claude-code
/plugin marketplace add VoltAgent/awesome-claude-code-subagents
/plugin marketplace add ccplugins/awesome-claude-code-plugins
```

### Add Key MCP Servers
```bash
# GitHub integration
/plugin install github-mcp-server

# Documentation
/plugin install context7-mcp

# Browser automation
/plugin install playwright-mcp-server
```

### Set Up CLAUDE.md
```bash
# In your project root
touch CLAUDE.md
# Add project-specific context, commands, and rules
```
